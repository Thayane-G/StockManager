# StockManager

Sistema web de gerenciamento de produtos desenvolvido com Flask no backend e HTML, CSS e JavaScript no frontend.

O projeto permite realizar operações completas de CRUD (criar, listar, editar e excluir produtos), além de busca e interface dinâmica.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos
- Busca em tempo real
- Interface web responsiva
- Integração com API REST

---

## Tecnologias utilizadas

### Backend
- Python
- Flask
- Flask-CORS
- SQLAlchemy
- MySQL

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## API Endpoints

| Método | Rota | Descrição |
|--------|------|----------|
| GET | /produtos | Lista todos os produtos |
| POST | /produtos | Cria um produto |
| PUT | /produtos/{id} | Atualiza um produto |
| DELETE | /produtos/{id} | Remove um produto |
